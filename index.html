Errors and Exceptions

What's the difference? Errors are problems that cause a program to stop execution, while Exceptions are raised when internal events change the normal flow of a program.

Syntax Errors

Occur when the proper syntax of Python is not followed. They prevent the program from running.

Logical Errors (Exceptions)

Occur during runtime after passing the syntax test. They disrupt normal program flow.

Syntax Error Example

Syntax Error Example
# initialize the amount variable
amount = 10000

# check that You are eligible to
# purchase Dsa Self Paced or not
if(amount>2999)  # Missing colon (:)
    print("You are eligible to purchase Dsa Self Paced")

Error: The code above would result in a syntax error because the if statement is missing a colon : at the end.

Correct syntax: if(amount>2999):

Logical Error (Exception) Example

Logical Error Example
# initialize the variable
marks = 10000

# perform division with 0
a = marks / 0  # This will raise ZeroDivisionError

print(a)

Error: The code above would result in a ZeroDivisionError because division by zero is not allowed.

Exception Handling in Python

Try and Except Block

The code that might generate an exception is placed inside the try block, and the code to handle the exception is written in the except block.

try:
    # code that may cause exception
    numerator = 10
    denominator = 0
    result = numerator/denominator
    print(result)
except:
    print("Error: Denominator cannot be 0.")
finally:
    print("This is finally block.")
Output:
Error: Denominator cannot be 0.
This is finally block.

try block

The code that might cause an exception is placed here.

except block

Executes when an exception occurs in the try block.

finally block

Always executes, regardless of whether an exception occurred.

Optional blocks:

  • else block: Runs if the code in the try block executes without exceptions.
  • finally block: Always executes, regardless of whether an exception occurred.

Functions in Python

A function is a reusable block of code that performs a specific task. Functions help in breaking our program into smaller and modular chunks, making it more organized and manageable.

Types of Functions in Python

  • Standard library functions - Built-in functions in Python.
  • User-defined functions - Custom functions created based on specific requirements.

Function Declaration

Function Syntax
def function_name(arguments):
    # function body
    # statements...
    return value  # optional

Simple Function Example

def greet():
    print("Hello!")

# Calling the function
greet()
Output:
Hello!

Function with Arguments

# function with two arguments
def add_numbers(num1, num2):
    sum = num1 + num2
    print("Sum: ", sum)

# function call with two values
add_numbers(5, 4)
Output:
Sum: 9

Return Statement

The return statement sends a value back to the function caller. It's optional - if not included, the function returns None.

# function definition with return value
def find_square(num):
    result = num * num
    return result

# function call
square = find_square(3)
print('Square:', square)
Output:
Square: 9

Default Arguments

Python allows us to specify default values for function arguments. These values are used when the corresponding argument is omitted.

def add_numbers(a = 7, b = 8):
    sum = a + b
    print('Sum:', sum)

# function call with two arguments
add_numbers(2, 3)

# function call with one argument
add_numbers(a = 2)

# function call with no arguments
add_numbers()
Output:
Sum: 5
Sum: 10
Sum: 15

Variable Length Arguments

*args (Non-Keyword Arguments)

Allows you to pass a variable number of non-keyword arguments to a function.

def adder(*num):
    sum = 0
    for n in num:
        sum = sum + n
    print("Sum:", sum)

adder(3, 5)
adder(4, 5, 6, 7)
adder(1, 2, 3, 5, 6)
Output:
Sum: 8
Sum: 22
Sum: 17

**kwargs (Keyword Arguments)

Allows you to pass a variable number of keyword arguments to a function.

def intro(**data):
    print("\nData type of argument:", type(data))
    for key, value in data.items():
        print("{} is {}".format(key, value))

intro(Firstname="Sita", Lastname="Sharma", 
      Age=22, Phone=1234567890)
Output:
Data type of argument: <class 'dict'>
Firstname is Sita
Lastname is Sharma
Age is 22
Phone is 1234567890

Recursive Functions

A recursive function is a function that calls itself during its execution. This approach can solve complex problems by breaking them down into simpler ones.

def factorial(x):
    """This is a recursive function
    to find the factorial of an integer"""
    if x == 1:
        return 1
    else:
        return (x * factorial(x-1))

num = 3
print("The factorial of", num, "is", factorial(num))
Output:
The factorial of 3 is 6

Variable Scope

Local Variables

Defined inside a function and can only be accessed within that function.

Global Variables

Defined outside a function and can be accessed both inside and outside functions.

Nonlocal Variables

Used in nested functions to refer to variables in the parent function's scope.

Example of Variable Scopes
# Global variable
message = 'Hello'

def outer():
    # Local variable for outer()
    message = 'local'
    
    # Nested function
    def inner():
        # Declare nonlocal variable
        nonlocal message
        message = 'nonlocal'
        print("inner:", message)
    
    inner()
    print("outer:", message)

outer()
print("Global:", message)
Output:
inner: nonlocal
outer: nonlocal
Global: Hello

Strings in Python

A string is a sequence of characters. In Python, strings are immutable, meaning they cannot be changed after creation.

Creating Strings

Using Single or Double Quotes

# create strings
name = "Python"
print(name)

message = 'I love Python.'
print(message)
Output:
Python
I love Python.

Multi-line Strings (Triple Quotes)

message = """
Never gonna give you up
Never gonna let you down
"""

print(message)
Output:
Never gonna give you up
Never gonna let you down

Accessing String Characters

Indexing

Access characters using index values, starting from 0.

greet = 'hello'
# access 1st index element
print(greet[1])
Output: e
Negative Indexing

Access characters from the end, starting with -1.

greet = 'hello'
# Access 4th last element
print(greet[-4])
Output: e
Slicing

Access a range of characters using the colon operator.

greet = 'Hello'
# From 1st to 3rd index
print(greet[1:4])
Output: ell

Escape Sequences

Escape sequences allow you to include special characters in strings.

# escape double quotes
example = "He said, \"What's there?\""

# escape single quotes
example = 'He said, "What\'s there?"'

print(example)
Output:
He said, "What's there?"

\'Single quote
\"Double quote
\aASCII Bell
\bASCII Backspace
\fASCII Formfeed
\nASCII Linefeed
\rASCII Carriage Return
\tASCII Horizontal Tab
\vASCII Vertical Tab
\oooCharacter with octal value ooo
\xHHCharacter with hexadecimal value HH

f-String Example (Python 3.6+)
name = 'Cathy'

print(f'{name} is from {country}')

Cathy is from UK

Raw Strings and Unicode

Raw Strings

Raw strings ignore escape sequences by prefixing the string with 'r'.

# Define raw string
s1 = r'Python\nis\easy\to\learn'
print(s1)
Output:
Python\nis\easy\to\learn
